# トータルアカデミー

パワーリフティングを学びながらPRを狙うための、スマホ対応PWA試作アプリです。

## できること

- 複数選手の切り替え
- 選手の追加・削除
- 性別、体重、体重階級、都道府県、試合予定の管理
- 所属エリアからJPA加盟都道府県協会リンクへ案内
- BIG3、補助種目、コンディショニング種目の記録
- 重量、回数、セット、RPE、メモの保存
- 必要な時だけセットごとの重量・回数・RPEを詳細記録
- 種目別e1RM、直近ボリューム、平均RPEの分析
- RPE、ボリューム、BIG3バランスに基づく「アカデミーからの評価」
- BIG3 / ベンチプレスのみのプラン対象選択
- トータルアカデミー式、HPS、5/3/1、Smolov Jr.の簡略テンプレート選択
- 週数、頻度、補助種目量、重点種目のカスタム
- 性別、体重階級、現在トータルを使った予測PRレンジ補正
- 設備依存種目の代替案表示
- プラン上での実績入力とRPE補正提案
- 前週のRPE実績を翌週のプランにメモとして反映
- 基礎知識タブでRPE、RIR、e1RM、Training Maxなどを説明
- IPF技術規則とJPAルールブック・通達への公式リンク
- バックアップJSONの書き出し・読み込みで端末データを保護
- 記録ログと12週プランのExcel書き出し

## 開き方

まずは `index.html` をブラウザで開くと使えます。

PWAとしてスマホのホーム画面に追加したい場合は、作業フォルダで次を実行してからブラウザで開きます。

```powershell
node server.mjs
```

URL:

```text
http://127.0.0.1:4173/
```

## 友人への共有

スマホやiPhoneでアプリのように使ってもらう場合は、PWAとしてHTTPSのURLで公開します。
おすすめは GitHub Pages、Netlify、Vercel のような静的サイトホスティングです。

iPhoneでの追加手順:

1. Safariで公開URLを開く
2. 共有ボタンを押す
3. 「ホーム画面に追加」を選ぶ
4. ホーム画面のトータルアカデミーアイコンから開く

記録データは各ユーザーの端末内に保存されます。友人と自動同期される形ではありません。
大きなアップデート前は、履歴タブのデータ管理からバックアップを書き出しておくと安心です。

## プラン設計メモ

- トータルアカデミー式は、RPEを学びながらPRを狙う標準プランです。
- 予測PRは、現在1RM、サイクル長、週頻度、種目別の伸びやすさを使った保守的なレンジです。
- 週頻度は単純な加点ではなく、重量を分散しやすい利点と疲労管理の難しさを両方見ます。
- HPS、5/3/1、Smolov Jr.は各メソッドの考え方を参考にした簡略テンプレートです。公式プログラムの完全再現、公式提携、公式承認を示すものではありません。

## 分析コメントの考え方

- アカデミー評価は診断ではなく、RPE、直近ボリューム、記録頻度、BIG3バランスから次の練習判断を助けるための目安です。
- RPEが予定より高い場合は、疲労、回復、技術の乱れなど複数の要因が考えられるため、重量より予定RPEを守ることを優先するコメントにしています。
- 参考: Helms et al. 2018 "RPE vs. Percentage 1RM Loading in Periodized Programs"; Helms et al. 2018 "Rating of Perceived Exertion as a Method of Volume Autoregulation"; Helms et al. 2020 "Methods for Regulating and Monitoring Resistance Training".
