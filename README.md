# Platform Buddy

MAX更新から大会挑戦まで、1人で始めたリフターを支えるスマホ対応PWA試作アプリです。

## できること

- 複数選手の切り替え
- 選手の追加・削除
- 性別、体重、体重階級、都道府県、試合予定の管理
- 所属エリアからJPA加盟都道府県協会リンクへ案内
- 初期表示をプランにしたPRサイクル中心の導線
- トップの「はじめにカード」から、LOG / PLAN / MEETの目的別ガイドを表示
- はじめてガイドを折りたたみ式にして、スマホ初期表示の情報量を軽量化
- メイン導線をLOG / PLAN / MAX / MEETの4タブに整理
- メイン導線はPLAN / LOG / MAX / MEETの順に配置
- PRサイクル設計から重複していた「現在の週」入力欄を削除し、週移動は前週/次週ボタンに統一
- ガイド表示を「はじめて」、説明を減らす表示を「すっきり」として切り替え
- プラン外の練習を残す自由記録
- BIG3、補助種目、コンディショニング種目の記録
- 重量、回数、セット、RPE、メモの保存
- セット数に応じたセット表で重量・回数・RPEを記録
- Set1を下のセットへコピーして素早く複数セット入力
- プラン実績と自由記録をまとめた種目別e1RM、直近ボリューム、平均RPEの分析
- RPE、ボリューム、BIG3バランスに基づくBuddyコメント
- BIG3 / ベンチプレスのみのプラン対象選択
- Buddy式、HPS、5/3/1、Smolov Jr.の簡略テンプレート選択
- プラン方式内でBuddyメソッド Lv1（標準）/ Lv2（実戦寄り）を選択
- Lv2ではRPEと%1RMを併用し、SQ/DL週2回、BP週3〜4回を目安に週内非線形で配置
- Buddy式で経験レベルに応じた現在地チェックを表示
- Buddy式で現在地チェック前の休養ブロックを表示
- Buddy式の最終週をBIG3大会想定MAXチェックとして表示
- 週数、頻度、補助種目量、重点種目のカスタム
- 性別、体重階級、現在トータルを使った予測MAXレンジ補正
- 設備依存種目の代替案表示
- プラン上での実績入力とRPE補正提案
- 前週のRPE実績を翌週のプランにメモとして反映
- ガイドON/OFFで説明量を切り替え、スッキリ表示では重要な警告とメニューを中心に表示
- プロフィール、PRサイクル設計、設備依存種目を記号ボタンで折りたたみ
- 基礎知識タブでRPE、RIR、e1RM、Training Max、現在地チェック、試技申請、検量、疲労管理などを説明
- 基礎知識タブ内の「白判定クイズ」で6カテゴリ合計30問の3択ルール学習
- MEETタブから白判定クイズと公式ルールへショートカット
- クイズの即時正誤判定、解説、公式参照リンク、最終確認日の表示
- クイズ進捗、正解数、間違えた問題の復習を端末内に保存
- IPF技術規則とJPAルールブック・通達への公式リンク
- バックアップJSONの書き出し・読み込みで端末データを保護
- 記録ログとプランのExcel書き出し

## Buddy式プランの流れ

- W1〜W4: 蓄積期。現在のMAXをより確実に挙げるためのフォーム再現性、RPE感覚、練習量を作る。
- Lv1: RPE習得と安全重視。初心者〜初中級者が壊れず進む標準設計。
- Lv2: 中級者以上向け。蓄積期から%1RMレンジと高重量シングルを少量使い、週内で強弱を作る。
- 休養ブロック: 現在地チェック前に3日程度の休養を入れる。
- W5: 現在地チェック。初心者は3〜5回@8、中級者はシングル@8〜9、上級者はシングル@9目安。
- 後半: 強化期からピーキングへ進み、重量に慣れながら疲労を管理する。
- 最終週: 通常練習ではなく、SQ→BP→DLを1日にまとめる大会想定MAXチェック。

10週プランは想定完了期間が約11週、12週プランは約13週です。

## 友人への共有

スマホやiPhoneでアプリのように使ってもらう場合は、PWAとしてHTTPSのURLで公開します。
iPhoneではSafariで公開URLを開き、共有ボタンから「ホーム画面に追加」を選びます。

記録データは各ユーザーの端末内に保存されます。友人と自動同期される形ではありません。
大きなアップデート前は、履歴タブのデータ管理からバックアップを書き出しておくと安心です。

## 分析コメントの考え方

- Buddyコメントは診断ではなく、RPE、直近ボリューム、記録頻度、BIG3バランスから次の練習判断を助けるための目安です。
- RPEが予定より高い場合は、疲労、回復、技術の乱れなど複数の要因が考えられるため、重量より予定RPEを守ることを優先するコメントにしています。
- 参考: Helms et al. 2018 "RPE vs. Percentage 1RM Loading in Periodized Programs"; Helms et al. 2018 "Rating of Perceived Exertion as a Method of Volume Autoregulation"; Helms et al. 2020 "Methods for Regulating and Monitoring Resistance Training".
